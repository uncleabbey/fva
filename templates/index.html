Welcome To VGC Food Vendor Capstone Project

This is Venture Gardens Group(VGC) Internship capstone project for backend interns (Python/Django Stack) The link to Coresapi documentation is here or you can use Swagger documentation

How it Works

Registration

Register a new user using Customer Registration or Vendor Registration

After doing this, check your mail for instruction on setting your password or you could copy your unique_ref from response of
registration and paste it in the documentation('/auth/set/password/user/{unique_ref}') path. Email is easier

after setting the password, copy the token in the response and goto Swagger documentation
and paste the token inside the Authorize button to get access to protected routes

or quickly use the following test users for quick Login if you don't want to go through registration stress
